java 多态
===

##向上转型

	Shape s=new Circle();
> （Shape为基类）创建一个Circle对象，把这个引用赋给Shape

##综述
+ 面向对象三个特征：
	+ 封装：隐藏类的内部实现机制
	+ 继承：代码重用
	+ 多态：由于java只能单一继承，为了弥补单一继承限制

##过程
1. 使用父类类型的引用指向子类的对象
2. 如果子类重写了父类方法调用子类方法（动态链接），否则调用父类的
3. 多态值针对方法，对变量无效
